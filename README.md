# Лабораторная работа №12  
## Аутентификация и работа с токенами (JWT, OAuth)

### Цель работы
Изучить механизмы аутентификации, работу с токенами JWT, хранение токенов и защиту данных.

---

## Описание программы
Приложение — простая имитация аутентификации пользователя.  
Реализовано:

- Форма входа (email, пароль)
- Генерация тестового JWT-токена
- Сохранение токена в localStorage
- Автоматический вход при перезагрузке
- Выход (удаление токена)
- Отображение защищённой страницы
